New to IF?

Unsure how to get started in the world of text adventures? Try these web sites:

  • Starter links at IFWiki: links for IF newcomers. (Maintained by David Welbourn and the community)
  • Brass Lantern Beginner Resources: a collection of articles for IF newcomers. (Maintained by Stephen Granade)
  • Baf's Guide to the IF Archive: an indexed, searchable database of the games in the Archive, plus contributed reviews. (Maintained by Carl Muckenhoupt)
  • The Interactive Fiction Database: another database of IF games and reviews. (Maintained by Mike Roberts)

To Submit Files

Files may be submitted through a web form, or by FTP to the directory ftp://ftp.ifarchive.org/incoming/. See the IF Archive README.
